Técnicas de Engenharia Social Identificadas:
  • Impersonação de marca legítima: Usa logo, layout e estilo dos Correios oficiais
  • Criação de urgência: "SUA ENCOMENDA FOI TRIBUTADA!" em letras vermelhas
  • Pressão psicológica: Status "AGUARDANDO PAGAMENTO" com ícone de alerta
  • Localização falsa: Menciona agência específica (Nova Iguaçu) para parecer real
  • CTA enganosa: Botão azul destacado induz clique para domínio malicioso
  • Documentação falsa: Imagem de comprovante com QR code faz parecer oficial
  • Redirecionamento oculto: Links apontam para domínio pay.libiere-sua-entrega.online
Como Identificar:
  • URL real dos Correios: www.correios.com.br
  • Domínio suspeito: libiera-sua-entrega.online
